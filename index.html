Wikimon

Create an artificial intelligent pet that learns to chat through mirroring

View project on GitHub

   
   

Overview

Wikimon is a Python script which creates an artificial intelligent digital pet that learns through mirroring. Users can chat with an fully-conversant artificial intelligent virtual friend that learns and imitates previous chats in context.

Licensed under the MIT License - https://opensource.org/licenses/MIT

Features

  • Remembers conversation history, then learns by mirroring it
  • Works with any Linux operating system
  • Starts-up with an input of user's phone number
  • 3-step configuration wizard that requires less than 1 minute to set-up

System Requirements

  • Python 2.7
  • These instructions have been tested on Ubuntu 14.04.3 and 15.10.

Getting started

 

First, install yowsup by following these steps:

Clone jlguardi’s yowsup fork and install it:

$ git clone https://github.com/jlguardi/yowsup.git && cd yowsup
$ sudo apt-get install python-setuptools python-dev
$ sudo python setup.py install
$ cd ..

At the time of writing (31 March 2016), the official yowsup source code does not work. There is, however, a fork by jlguardi which does work. The following steps were successfully tested but please note that the jlguardi fork may not necessarily work in the future.

Register your phone number:

`$ git checkout Android_2.12.440
$ sudo ./yowsup-cli registration --requestcode sms --phone <phone number> --cc <country code> --mcc <mcc> --mnc <mnc>
$ sudo ./yowsup-cli registration --register <code sent to you via SMS> --phone <phone number> --cc <country code>

For the mobile country codes required by the two commands above, find them here.

After registration, configure and run the server:

$ git clone https://github.com/facerecog/wikimon.git && cd wikimon
$ sudo python setup.py install
$ sudo python src/server.py
